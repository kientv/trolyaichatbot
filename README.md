# Hướng dẫn sử dụng aichatbot.com.vn

**Nhân viên số AI đa kênh 24/7** - Tư vấn, Chăm sóc khách hàng, Chốt đơn tự động

### Xem hướng dẫn ở local

```bash
# Clone repo
git clone https://github.com/kientv/trolyaichatbot.git

# Chạy server local
cd trolyaichatbot
npx serve .
```

Mở trình duyệt: http://localhost:56507
